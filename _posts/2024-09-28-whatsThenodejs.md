# 1. nodejs란?

자바스크립트는 프론트 엔드에서 html과 함께 자주 사용되는 언어입니다.
<br>
따라서 백엔드를 구성하기 위해서는 자바스크립트 이외의 자바, 파이썬 등의 언어를 사용해야 합니다.
<br>

> ※ 처음 코딩 배우실 때 흔히 하는 착각이<br>`[자바스크립트 = 자바 + @]`니까 확장판이나 강화판 같은 거라고 생각합니다.<br>저도 처음에는 그랬습니다. 하지만 그 둘은 용도가 서로 다릅니다.

하지만 nodejs는 자바스크립트를 백엔드에서도 사용할 수 있도록 하는 런타임 환경으로, nodejs가 있다면 자바스크립트 하나로 프론트와 백을 모두 구현할 수 있습니다.

<br>

코딩을 요리에 비유한다면

- 자바스크립트 - 식재료
- nodejs - 주방
- vscode - 요리 도구(식칼, 프라이팬 등)

정도로 볼 수 있습니다.
<br><br>
하고자 하는 요리가 있다면, 그 레시피에 맞는 주 식재료가 정해져 있을 것이고, 그 요리를 하기에 알맞는 주방이 있을 겁니다. 대형 바베큐 그릴을 실내에서 사용할 수는 없는 것처럼요.
<br><br>
마찬가지로, 우리가 쓸 수 있는 식재료는 자바스크립트뿐이고, 그 자바스크립트를 조리할 수 있는 주방은 nodejs 한 곳 뿐이기 때문에 우리는 선택지가 없습니다.
<br><br>
하지만 식칼의 브랜드와 제품에는 선택지가 있고, 저는 그 중에서 vscode라는 식칼을 추천하는 것에 불과하기 때문에, 더 선호하시는 게 있으시다면 그걸 쓰시면 됩니다.

# 3. nodejs 파일 생성하기

<br>

## 3-1. 개발에 필요한 세팅하기

1. vscode를 실행합니다.<br><br>
2. `` ctrl + ` `` 을 눌러서 터미널을 열어줍니다.<br>(~) 기호를 입력할 때 쉬프트 버튼을 누르지 않으면 \`이 나옵니다.<br><br>
3. 터미널에 다음과 같이 입력합니다.<br>※ `|` 기호의 입력 방법입니다. 키보드의 엔터키 근처에 `￦` 버튼이 있을 겁니다.<br> `shift + ￦` 을 누르시면 됩니다.

```javascript
sudo apt-get install curl -y
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo apt-get install build-essential
sudo apt-get update
```

## 3-2. nodejs 파일 작성

<br>

1. 작업 파일을 저장할 폴더를 생성해주세요.<br><br>
2. vscode의 화면 상단에서 `파일 - 폴더 열기`를 눌러 해당 폴더를 열어주세요.<br><br>
3. 그 폴더 안에 `hello.js` 라는 이름의 파일을 생성합니다.<br>※ 파일 생성 방법입니다. 폴더를 열고 나서 vscode 화면 왼쪽에 보이는 것이 해당 폴더의 내부입니다. 거기서 `마우스 우클릭 - 새 파일` 을 누르거나 그 폴더 영역의 상단에 있는 `종이에 (+)가 붙은 아이콘` 을 누르시면 됩니다.<br><br>
4. 그리고 아래의 내용을 입력해주세요.

```typescript
const http = require("http"); // (1)

const server = http.createServer((req, res) => {
  // (2)
  res.setHeader("Content-Type", "text/plain"); // (3)
  res.write("hello\n"); // (4)
  res.end("Nice to meet you"); // (5)
});

server.listen(8000, () => console.log("Hello Node.js"));
--6;
```

> **주의!! `// (숫자)`는 어디에 대한 설명인지 표시하기 위한 책깔피 느낌으로 제가 넣은 것입니다. 실제 코드에는 이걸 빼고 입력하셔야 합니다.**

| 번호 | 설명                                                                                                                                                                                                                                                              |
| ---- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| (1)  | http 객체 생성<br>※ nodejs에서는 이미 만들어진 기능을 가져와서 코드 내에서 사용할 때는 이렇게 `require("...")` 을 씁니다.                                                                                                                                         |
| (2)  | 노드 서버를 객체 생성하면서, 서버가 요청을 받을 때마다 실행할 구체적인 내용을 정합니다.                                                                                                                                                                           |
| (3)  | 응답이 어떤 방식으로 돌아올 지 타입을 설정합니다.                                                                                                                                                                                                                 |
| (4)  | 클라이언트에게 "hello"라는 메시지를 보냅니다.                                                                                                                                                                                                                     |
| (5)  | 서버가 응답하면서 내보낼 메시지입니다.<br>※ 응답하면서 내보낼 메시지를 정할 필요는 없지만, 적어도 res.end()는 반드시 해줘야 합니다. 그렇지 않으면 영원히 응답하지 않고 무한 로딩에 빠집니다.                                                                      |
| (6)  | 서버가 8000번 포트에서 요청을 받도록 합니다.<br>그리고 서버가 시작될 때 콘솔창(vscode에서는 `터미널` 이라는 이름으로 표시될 겁니다.)에 입력될 메시지를 설정합니다.<br>※ 이미 포트가 사용 중이라면 오류가 뜰 수도 있습니다. 그럴 때는 다른 번호로 바꾸시면 됩니다. |

<br>

# 4. 서버 실행해보기

1. `` ctrl + ` `` 을 눌러서 터미널을 열어줍니다.<br><br>
2. 그리고 `node hello.js` 라고 입력해줍니다. 여기서 `hello.js` 는 방금 전에 만든 파일의 이름입니다.
   > 여기서 오류가 뜬다면, 그건 경로가 잘못 되었을 가능성이 가장 높습니다.<br><br>vscode의 좌측을 보시면 현재 파일이 들어 있는 폴더가 열려 있습니다. 그 폴더에서 `마우스 오른쪽 - 상대 경로 복사` 를 누르시면 경로를 복사할 수 있습니다.<br><br>그리고 터미널에 `cd + 복사한 경로`를 입력하시면 해당 폴더로 이동됩니다. 이제 다시 `node hello.js`를 입력해줍니다.<br>
3. 인터넷 창을 열어서 화면 상단 주소창에 `localhost:8000` 을 입력합니다.
   > 여기서 8000은 아까 설정한 포트 번호이므로, 그것을 수정하셨다면 이것도 그거에 맞춰서 수정해주시면 됩니다.
4. 터미널에서 `ctrl + c` 를 눌러서 현재 실행 중인 서버를 종료하는 것으로 마무리합니다.
