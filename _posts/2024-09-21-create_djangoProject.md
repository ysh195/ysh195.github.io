## 1-2. 프로그램 설치하기

<br>

다운로드 받은 파일들을 실행해서 설치하면 됩니다. 모두 확인 버튼만 계속 누르면 되니 어렵지 않습니다.

<br>
참고로, 아나콘다는 파일 자체가 워낙 커서 다운로드도 설치도 모두 오래 걸립니다.
<br>
오류나 렉 때문에 설치가 안 되는 것은 아닌지 의심이 되실 수 있지만, 그게 정상입니다.
<br><br>

### 단, 파이썬과 아나콘다를 설치할 때 명심해야 할 점이 있습니다.

<br>

- `파이썬의 경우`

설치할 때 화면 하단에
<br>

- [x] Use admin privileges when installing py.exe
      <br>
- [x] Add python.exe to PATH
      <br>
      를 체크해주세요. 굳이 체크하지 않아도 따로 설정하면 그만이지만, 클릭 한 번에 끝날 일을
      따로 조사까지 해가며 다시 진행할 필요는 없습니다.
      <br>
      <br>
      그리고 혹시 파이썬을 설치하다가 중간에 저절로 취소되는 일이 있다면 파이썬 설치 프로그램을 관리자 권한으로 실행해주세요.<br>

<br>

- `아나콘다의 경우`

설치할 때 화면 중단에
<br>

- [x] Add Anaconda3 to my PATH environment variable
      <br>
- [x] Remove Anaconda3 as my default Python \*.\*\*
      <br>
      를 체크해주세요. 파이썬의 경우와 같습니다. 클릭 한 번으로 끝내면 충분한 일입니다.

# 2. 프로젝트 생성하기

<br>

| 번호 | 설명                                                                                                                                                                                                                                                                                   |
| ---- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 1    | 파이참을 관리자 권한으로 실행합니다.                                                                                                                                                                                                                                                   |
| 2    | new project를 클릭합니다.                                                                                                                                                                                                                                                              |
| 2-1  | ![장고생성1](https://ysh195.github.io/images/24-09-21_post/create_django1.PNG) <br> django를 클릭 > baseconda 클릭 > select path 클릭 <br> name 부분에는 원하는 프로젝트 폴더의 이름을 입력합니다. <br> [] Create Git repository는 선택사항입니다.                                     |
| 2-2  | ![장고생성2](https://ysh195.github.io/images/24-09-21_post/create_django2.PNG) <br> 아나콘다가 설치된 폴더 안에 python.exe라는 파일이 있을 것입니다. (아나콘다 설치 경로는 각자 다릅니다.)그것을 선택하고 OK 버튼을 누릅니다.                                                          |
| 2-3  | create 버튼을 누릅니다.                                                                                                                                                                                                                                                                |
| 3    | ![장고생성3](https://ysh195.github.io/images/24-09-21_post/create_django3.PNG) <br> 파이참 화면 좌측 상단에 이런 아이콘이 있을 것입니다.<br> 없다면 아니면 그냥 File, Edit... 등이 나열되어 있을 수도 있습니다.<br>아이콘이 있다면 그것을 누른 후 File을 클릭. 아니면 바로 File을 클릭 |
| 4    | File을 눌러서 나온 항목들 중에서 settings를 클릭                                                                                                                                                                                                                                       |
| 5    | 좌측에 굵은 글씨로 Project: \[ 프로젝트명 \] 이라고 된 항목이 있을 것입니다. 그것을 클릭 후 python interpreter 클릭                                                                                                                                                                    |
| 6    | add interpreter 클릭 > add local interpreter 클릭                                                                                                                                                                                                                                      |
| 7    | Virtual Environment > Environment: New<br>(location은 아마 자동으로 설정되어 있을 겁니다. 안 되어 있으면 현재 프로젝트 폴더의 절대경로 뒤에 "\\venv"를 붙입니다.) <br> > Base interpreter: 아까 전의 아나콘다 python.exe 다시 선택                                                     |
| 8    | 그러면 pip과 setuptool이라는 항목이 생겼을 겁니다. 맨 아래의 ok 클릭                                                                                                                                                                                                                   |

<br>

혹시 setuptool이 없으셨다면 프로젝트 폴더에서
\[ ctrl + ` \] 을 눌러서 터미널을 실행시키고, 터미널에 다음과 같이 입력합니다.

```java
venv\Scripts\activate
pip install setuptools wheel
```

<br>

이것으로 프로젝트 생성 및 세팅이 끝났습니다.
